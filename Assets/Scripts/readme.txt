AppManager - главный класс, ответственный за работу всего приложения, за переключение персонажей, за загрузку настроек и т.д.
UIManager - класс, связывающий все UI элементы с AppManager.
CharacterManager - вспомогательный класс, хранящий информацию о всех персонажах, дает доступ к выбранному персонажу.
CharacterSelector - класс, контролирующий работу окна выбора персонажа.
ModeSwitcher - класс, контролирующий работу выпадающей кнопки, ответственной за выбор режима.
NetworkManager - класс, в котором описано подключение к "API" расписания ИКТИБ.
ScheduleGenerator - класс, генерирующий разметку из UI элементов Unity на основе JSON полученного от NetworkManager, для отрисовки расписания.
CameraFocusController - вспомогательный класс, необходим для работы автофокуса камеры устройства.
